Notes

Here I record some of the notes I take when tweaking this app.

For routing, I have found there are two ways.

  1. Manual routing where I create a module, declare/import its component, expose the module to the root AppModule by stating it under AppModule's "imports" section.
  2. The cli command ng g m notes --route notes --module app.module which puts the routing straight in the app.module but using a code that paths it straight to the string immediately after path (path: 'destination')
